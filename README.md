### Required Components
1. Arduino Uno Board with USB connector
2. Breadboard
3. LCD Screen (compatible with Hitachi HD44780 driver)
4. LDR(Light-Dependent Resistor)/Photocell
5. LED
6. 220 ohm resistor x 4
7. Jumper wires x 30
